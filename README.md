## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install elr-scss-articles
```

or

```sh
yarn add elr-scss-articles
```

## Implementation

### Scss

```scss
article {
  @include elr-article;
}
```
